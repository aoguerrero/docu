Run a new container
-------------------

$ docker run --volume {host dir}:{container dir} --name {} -d -p {host port}:{container port} -e {environment variables} image

Example:

$ sudo docker run --volume ~/docker-shared:/mnt/shared --name mint-db-thy -d -p 1521:1521 -p 5500:5500 andresguerrero/oracledb:18.4.0

Useful commands
---------------

$ docker exec -it {id} /bin/bash
$ docker commit {id} {image name}
$ docker image tag {id} {new tag}
